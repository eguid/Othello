# Othello
Othello game（奥赛罗棋、又叫翻转棋、黑白棋），自娱自乐作品。用行动证明，要啥自行车，有画布画笔就能做出游戏！

## 运行游戏
执行App.main()运行游戏

## 基于
Based on javafx,graphic using canvas,support jdk8+
